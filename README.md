## Graphing Calculator OS 
*An operating system made from scratch.*
 - The user will be able to type in mathematical equations and see them drawn on the screen. 
 - The OS will handle input from the keyboard, interpret the equation, and convert it into points that can be plotted as a graph. 
 - It will display axes and a grid, and allow the user to view how the function behaves visually instead of just numerically. 
 - The OS will support basic math operations at first. 

Made for Hack Club Boot!
